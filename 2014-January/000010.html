hi!

folgende idee als zukünftige dm4 erweiterung:
die möglichkeit, einen simplen kreis bzw. radar einzublenden. 
damit könnte man die beziehung einer idee zu einem zentralen gedanken visualisieren.

bei mir würde das z.b. so aussehen:
in der mitte der begriff 'zeit'. weit aussen, die idee der 'vertikalen zeit'. ganz nahe am zentrum die idee 'zeit ist eine illusion'.
später könnte man anhand des abstandes auch eine gewichtung berechnen (im innersten kreis = hot; weiter weg = 'von weit hergeholt' ;)

bei kommerziellen produkten hab ich so eine option sehr selten gefunden (im anhang ein bild).
umsetzung stelle ich mir vor: entweder eine zusätzliche topic-map type (topic-map, geo-map oder circle) oder ein plugin.

was mein ihr? habt ihr keinen bedarf an sowas?
